# Personal Assistant Project

## Goal
Build a private executive personal assistant for managing notes, tasks, follow-ups, partner updates, and draft communications.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- SQLite for local storage
- Future integrations: Gmail, Google Calendar, Google Drive, CRM

## Assistant Behavior
- Be concise and business-oriented.
- Preserve professional tone in drafted emails.
- Always show action items clearly.
- Never delete data without confirmation.
- Never expose secrets or API keys.
- Use environment variables for credentials.

## Core Features
1. Dashboard
2. Task tracker
3. Notes repository
4. Follow-up tracker
5. Email draft generator
6. Weekly status summary generator
7. Search across notes and tasks

## Coding Rules
- Keep components small.
- Use clear names.
- Add error handling.
- Avoid overengineering.
